Use this typealias for serializable classes.

Provided for convenience so that not every type needs to have the proper @Serializable(...) annotation.

The java.util.UUID is not serializable by default by the KotlinX Serialization package, so in order to use the UUID type at the REST API level declaring a custom serializer type is required.
