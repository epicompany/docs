This type defines a JSON Web Signature (JWS) data structure in its compact serialization format according to RFC 7515. A JWS is a JSON object that contains an arbitrary payload, a cryptographic signature over that payload, and a metadata header describing the cryptographic method used to compute and verify the signature. It has the following format:

BASE64URL || '.' || BASE64URL || '.' || BASE64URL
